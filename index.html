Instruções:

  1. Copie a URL de uma imagem do Instagram;
  2. Cole na caixa de texto;
  3. Você será redirecionado a uma página com a imagem, clique com o direito e selecione salvar imagem.
    4. RESTRIÇÕES: Não funcionará com vídeos e álbuns (post único com mais de uma imagem) somente a primeira foto desta postagem.

Como funciona:

Esta é a maneira mais simples de se obter uma imagem do Instagram em qualidade total, pode ser feita manualmente, o que este site faz é automatizar este processo de copiar e colar.

A URL dA foto possui um código e o nome do usuário marcados com "***" em negrito abaixo:

https://instagram.com/p/***/?hl=pt-br&taken-by-***

O que este site faz é pegar este código e colá-lo na área também demarcada na URL abaixo:

https://instagram.com/p/***/media/?size=l

Este novo link te leva até a foto no tamanho em que foi publicada.
